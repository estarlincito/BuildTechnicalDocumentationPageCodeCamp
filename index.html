¿Qué es HTML?

  • HTML significa Hyper Text Markup Language
  • HTML es el lenguaje de marcado estándar para crear páginas web
  • HTML describe la estructura de una página web
  • HTML consta de una serie de elementos
  • Los elementos HTML le dicen al navegador cómo mostrar el contenido
  • Los elementos HTML etiquetan piezas de contenido como "esto es un encabezado", "esto es un párrafo", "esto es un enlace", etc.

    • <!DOCTYPE html>

    La declaración <!DOCTYPE html> define que este documento es un documento HTML5

    <!DOCTYPE html>

    html

    El elemento <html> es el elemento raíz de una página HTML

    <html> (etiqueta de inicio)
    </html> (etiqueta final)

    head

    El elemento <head> contiene metainformación sobre la página HTML

    <head> (etiqueta de inicio)
    </head> (etiqueta final)

    title

    El elemento <title> especifica un título para la página HTML (que se muestra en la barra de título del navegador o en la pestaña de la página)

    <title> (etiqueta de inicio)
    </title> (etiqueta final)

    body

    El elemento <body> define el cuerpo del documento y es un contenedor para todos los contenidos visibles, como encabezados, párrafos, imágenes, hipervínculos, tablas, listas, etc.

    <body> (etiqueta de inicio)
    </body> (etiqueta final)

    div

    La etiqueta <div> define una división o una sección en un documento HTML.

    La etiqueta <div> se utiliza como contenedor de elementos HTML, que luego se estiliza con CSS o se manipula con JavaScript.

    La etiqueta <div> se puede diseñar fácilmente utilizando el atributo class o id.

    ¡Se puede poner cualquier tipo de contenido dentro de la etiqueta <div>!

    Nota: De forma predeterminada, los navegadores siempre colocan un salto de línea antes y después del elemento

    .

    <div> (etiqueta de inicio)
    </div> (etiqueta final)

    a

    La etiqueta <a> define un hipervínculo, que se utiliza para vincular de una página a otra.

    El atributo más importante del elemento <a> es el atributo href, que indica el destino del enlace.

    <a href="link">

    ul

    La etiqueta <ul> define una lista desordenada (con viñetas).

    Utilice la etiqueta <ul> junto con la etiqueta <li> para crear listas desordenadas.

    <li> (etiqueta de inicio)
    </li> (etiqueta final)

    li

    La etiqueta <li> define un elemento de lista.

    <li> (etiqueta de inicio)
    </li> (etiqueta final)

    nav

    La etiqueta <nav> define un conjunto de enlaces de navegación.

    Tenga en cuenta que NO todos los enlaces de un documento deben estar dentro de un elemento <nav>. El elemento <nav> está diseñado solo para el bloque principal de enlaces de navegación.

    <nav> (etiqueta de inicio)
    </nav> (etiqueta final)

    footer

    La etiqueta <footer> define un pie de página para un documento o sección.

    Un elemento <footer> normalmente contiene:


  • información de autoría
  • informacion registrada
  • Información del contacto
  • mapa del sitio
  • volver a los enlaces superiores
  • documentos relacionados

    • Puede tener varios elementos <footer> en un documento.

    <footer> (etiqueta de inicio)
    </footer> (etiqueta final)
